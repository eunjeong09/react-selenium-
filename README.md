#### 파일 구조는 아래와 같습니다. 프로젝트 clone전에 꼭 참고해주세요!
- react-selenium-basic
  - react : 카운터를 출력하는 react app 
  - selenium 
    - main.py : 자동화 테스트를 위한 selenium, 카카오 메시지 보내기 기능 포함
